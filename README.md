 # Telecom Customer Churn Analysis (EDA Project)

## Project Overview

This is an Exploratory Data Analysis (EDA) project focused on understanding customer churn in the telecom domain.

The dataset is sourced from Kaggle and includes customer demographics, service usage, contract details, and churn status.

The objective is to identify patterns and factors that influence customer churn using visual and descriptive analysis.

-----------------------------------------------------------------------------------------------------------------------------------

## Key Insights

### --Tenure

Customers with tenure under 12 months show the highest churn (40%+).

Churn decreases with tenure and drops below 15% after 24+ months.

### --Service Subscriptions

### --Internet Service Type

Fiber optic users show higher churn (~35‚Äì40%) than DSL users (~15‚Äì20%).

Customers with no internet service show the lowest churn (<10%).

### --Contract & Billing

Month-to-month contracts have the highest churn (40%+).

One- and two-year contracts significantly reduce churn (<15%).

Electronic check users churn more than customers using automated payment methods.
